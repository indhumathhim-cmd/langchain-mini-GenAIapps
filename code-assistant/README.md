# Code Assistant

Streamlit app that generates coding help using LangChain + OpenAI.

## Setup

```powershell
cd "C:\Users\USER\Desktop\Tarot Cards _RAG files\Langchain_Folder\code-assistant"
python -m pip install -r requirements.txt
```

## Environment Variables

1. Copy `.env.example` to `.env`.
2. Set your key:

```env
OPENAI_API_KEY=your_openai_api_key_here
```

## Run

```powershell
python -m streamlit run app.py
```
