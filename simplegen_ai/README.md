# SimpleGen AI

Terminal-based LangChain OpenAI prompt app.

## Setup

```powershell
cd "C:\Users\USER\Desktop\Tarot Cards _RAG files\Langchain_Folder\simplegen_ai"
python -m pip install -r requirements.txt
```

## Environment Variables

1. Copy `.env.example` to `.env`.
2. Set your key:

```env
Open_API_Key=your_openai_api_key_here
```

## Run

```powershell
python app.py
```
